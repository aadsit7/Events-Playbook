Partner Portal / Event Workspace
Event details sync live from the Events tab of the Partner Portal Google Sheet — use “Change event” to switch (the new event’s password is required if it has one). Each event opens with only its own contacts: upload a target list and it’s saved to the Event_Contacts tab of the sheet, then reloaded automatically the next time you open the event. Playbook checklist progress is a local working copy.
Open your event
Enter your event password to open it. If one password unlocks several events, you’ll pick the right one from the list.
Only that event’s details load — contacts stay as they are.
